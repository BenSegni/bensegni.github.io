This article dives into using parameters with SASS mixins to craft flexible style blocks. We'll explore a common approach to creating background images using a @mixin as our learning ground. Let's dissect its construction!

  
    @mixin background-image-config($name, $size, $no-repeat: true) {
      background-image: url('../../img/#{$name}');
      background-attachment: fixed;
      background-size: $size;

      @if($no-repeat) {
        background-repeat: no-repeat;
      }
    }

The standard background image css might look a little like this: 

  
    .background-image {
      background-image: url('../../img/some_image.jpg');
      background-attachment: fixed;
      background-size: 100vw;
      background-repeat: no-repeat;
    }

For this example, we've chosen to make the image, name, size, and repetition behavior flexible. 

  
    @mixin background-image-config($name, $size, $no-repeat: true) {
      ...
    }
These are defined by using $ and then appending the variable name. 
  
    @mixin background-image-config($name, $size, $no-repeat: true) {
      background-image: url('../../img/#{$name}');
      background-attachment: fixed;
      background-size: $size;

      @if($no-repeat) {
        background-repeat: no-repeat;
      }
    }
Now the @mixin is created, it just needs to be dropped into a class selector: 
  
    background-image-one {.
      @include background-image-config('background_one.svg', 100%, true);
    }

    .background-image-two {
      @include background-image-config('background_two.svg', 50%, false);
    }

    .background-image-three {
      @include background-image-config('background_three.svg', 75%, true);
    }
the .background-image-one class will then output the following when check in dev tools: 
  
    .background-image-one {
      background-image: url('../../img/background_one.svg');
      background-attachment: fixed;
      background-size: 100%;
      background-repeat: no-repeat;
    }
There you go, flexible SASS which can be applied to lots of different edge cases.
